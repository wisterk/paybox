+ Создать платёж

Платёж создаётся без метода — пользователь выберет способ оплаты.

Плейсхолдеры: {payment_id} {order_id} {status} {amount} {currency}

Открыть страницу оплаты

/ API Endpoints

POST /api/v1/payments
Создать платёж. Вернёт paymentUrl для редиректа.
GET /api/v1/payments/{id}
Информация о платеже по ID.
GET /api/v1/payments/by-order/{orderId}
Список платежей по номеру заказа.
POST /api/v1/payments/{id}/refresh
Обновить статус от провайдера.
GET /api/v1/payments/methods
Доступные методы оплаты.
GET /api/v1/rates
Курсы криптовалют к RUB.

$ curl

curl -X POST /api/v1/payments \ -H "Content-Type: application/json" \ -H "X-API-Key: YOUR_API_KEY" \ -d '{"amount": 500, "currency": "RUB", "orderId": "ORDER-123", "description": "Payment for services"}'

Ответ содержит paymentUrl — перенаправьте пользователя по этой ссылке.
